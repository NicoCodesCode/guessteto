# Guessteto 🎯

A simple command-line number guessing game written in Python. The program picks a random number between 1 and 100 and challenges you to guess it within a limited number of attempts based on your chosen difficulty.

Project made for: https://roadmap.sh/projects/number-guessing-game

## How to Play

1. Run the script with Python
2. Select a difficulty level
3. Guess the number — the game will tell you if the answer is higher or lower
4. Win by guessing correctly before you run out of chances!

## Difficulty Levels

| Level  | Chances |
| ------ | ------- |
| Easy   | 10      |
| Medium | 5       |
| Hard   | 3       |

## Requirements

- Python 3.x

## Running the Game

```bash
python main.py
```

## Features

- 3 difficulty levels
- Input validation throughout
- Play again option after each round
